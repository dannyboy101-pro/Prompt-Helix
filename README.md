# Prompt-Helix
Prompt Helix is a browser extension that enhances web pages using AI




PRIVACY POLICY

Privacy Policy for Prompt Helix

Prompt Helix collects webpage content and user-provided prompts only when the user explicitly activates the extension.

This data is used solely to generate AI-powered analysis and responses. Prompt Helix does not monitor browsing activity, track users across websites, or collect data without user action.

Email addresses are collected for account identification and subscription management only.

Payments are processed securely by Stripe. Prompt Helix does not store or process payment card details.

User data is not sold or shared for advertising or marketing purposes.

For questions, contact: helixlabs.support@gmail.com
